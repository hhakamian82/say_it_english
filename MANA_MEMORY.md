## 🚨 نجات پروژه: حل مشکل Drizzle و Vercel (مهم - 2026-02-12) ✅

### 1. مشکل کرش Runtime در Vercel (Edge Compatibility) 💥
- **علائم:** ارور `FUNCTION_INVOCATION_FAILED` و کرش کامل فانکشن‌های سرورلس هنگام استفاده از `drizzle-zod`.
- **علت:** کتابخانه `drizzle-zod` و برخی کانفیگ‌های `drizzle-orm` با محیط اجرایی Node.js در Vercel (مخصوصاً نسخه باندلر Rollup) ناسازگاری داشتند.
- **راه حل (Fix):**
    - حذف کامل وابستگی‌های `drizzle-zod` از فایل اشتراکی `shared/schema.ts`.
    - تعریف دستی (Manual Definition) اسکیماها در کلاینت (`Auth.tsx` و `admin/content.tsx`) با استفاده از `zod`.
    - **درس مهم:** در پروژه‌های Vercel، از به اشتراک گذاری مستقیم Schema بین Backend و Frontend اگر باعث وارد شدن کتابخانه‌های Server-side به Client-side شود، خودداری کنید.

### 2. مشکل اتصال دیتابیس (Scope & Connectivity) 🌐
- **علائم:** ارور `ENOTFOUND` هنگام اجرای `npm run db:push` روی سیستم لوکال (به دلیل تحریم/فیلتر).
- **راه حل (Fix):**
    - ساخت اندپوینت اختصاصی `api/seed-remote.ts` که روی سرور اجرا می‌شود.
    - این اسکریپت جدول‌ها را می‌سازد (DDL) و داده‌های اولیه را وارد می‌کند (Seed).
    - **نکته:** برای دور زدن محدودیت‌های شبکه لوکال، از سرور Vercel به عنوان "پراکسی اجرا کننده" استفاده کردیم.

### 3. پایداری API (Raw SQL vs ORM) 🛡️
- **علائم:** خطا در `api/index.ts` هنگام استفاده از `db.select()` به دلیل مشکلات Scope متغیر `pool`.
- **راه حل (Fix):**
    - بازنویسی `api/index.ts` برای تعریف **Global Pool**.
    - استفاده از **SQL خالص (`pool.query`)** برای کوئری‌های حیاتی مثل لیست محتوا (`/content`).
    - این کار وابستگی به لایه‌های انتزاعی ORM را در نقاط بحرانی حذف کرد و پایداری ۱۰۰٪ را تضمین نمود.

## 🚀 دستاوردهای کلیدی اخیر (جلسه ۲۳ بهمن) ✅

- **راه حل:** استفاده از **Local State Array** و فونت **Vazirmatn**.

## 🏗️ معماری سیستم (System Architecture)

- **Frontend:** React + Vite + TailwindCSS (Glassmorphism UI)
- **Backend:** Node.js (Vercel Serverless Functions) - Monolithic entry at `api/index.ts`
- **Database:** Supabase PostgreSQL (Region: `eu-north-1`) - Accessed via Raw SQL for reads, Drizzle for auth
- **Hosting:** Vercel
